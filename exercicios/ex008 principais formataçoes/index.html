Principais Formataçoes

Negrito / Destaque

nesta frase temos um termo em negrito usando a tag <b> (nao semantica)

nesta frase tenho um termo em destaque usando a tag <strong> (semantica)

Italico / enfase

nesta fase temos um termo em italico usando a tag <i> (nao semantico)

nesta frase temos um termo em enfase usando a tag <em> (semantico)

Texto marcado

podemos criar tambem um texto marcado usando a tag <mark>, e para mudar sua cor é só usar o style="background-color: (cor que voce quer usar);" dentro da tag <mark>

se voce quiser trocar a cor padrao do mark, é só usar o mark {background-color: (cor que voce quer usar);} no head dentro de um style que ele muda a cor padrao

Texto pequeno

estamos criando um texto pequeno (a tag é <small>) neste paragrafo. (a tag <small> é semantico pois letras pequenas podem dar significado as palavras como os contratos quando tem as letras pequenas)

Texto deletado

Podemos marcar um texto como excluido (tag <del>) que ele deve ser lido, mas não considerado

texto inserido

podemos marcar um texto como inserido (tag <ins>) para dar uma enfase e indicar que ele foi adicionado depois

tambem podemos sublinhar usando a tag <u> mas ela NÃO É SEMANTICA

texto sobrescrito

para inserir coisas do tipo x20 + 3

usando a tag <sup>

texto subscrito

para inserir coisas do tipo H2O

usando a tag <sub>
